סדר העץ $\unicode{x297D}$ בסבך

הגדרה נניח $\nu_0, \nu_1 \in S^1$.

  • שיכון אלמנטרי $\pi : J_{\nu_0} \to J_{\nu_1}$ נקרא יפה אם $\nu_0 < \nu_1$ ו-$\pi(p_{\nu_0}) = p_{\nu_1}$.

  • $\nu_0 \unicode{x297D} \nu_1$ אם יש שיכון יפה $\pi : J_{\nu_0} \to J_{\nu_1}$. נסכים ש-$\pi(\nu_0) = \nu_1$.

  • אם $\nu_0 \unicode{x297D} \nu_1$ אז נקרא לשיכון היפה $\pi_{\nu_0,\nu_1} : J_{\nu_0} \to J_{\nu_1}$. מיד נראה שאם יש שיכון יפה אז הוא יחיד ולכן אין דילמה איזה שיכון לבחור אם יש כמה.
