M1

טענה

  1. אם $\pi$ מעיד ש-$\nu_0 \unicode{x297D} \nu_1$ אז $\pi \mid \alpha_{\nu_0} = \text{id}$ ו-$\pi(\alpha_{\nu_0}) = \alpha_{\nu_1}$. (ולכן $\pi$ הוא העד היחיד.)

  2. $\unicode{x297D}$ הוא סדר עץ על $S^1$ כך שאם $\nu_0 \unicode{x297D} \nu_1$ אז $\alpha_{\nu_0} < \alpha_{\nu_1}$.

  3. $\langle \pi_{\nu_0, \nu_1} \mid \nu_0 \unicode{x297D} \nu_1 \rangle$ היא מערכת קומוטטיבית. $\langle \pi_{\nu_0, \nu_1}| \nu_0 + 1 \mid \nu_0 \unicode{x297D} \nu_1 \rangle$ היא מערכת קומוטטיבית של פונקציות שומרות סדר.

  4. נקבע $\alpha_i = \alpha_{\nu_i}$ ו-$S_i = S_{\alpha_i} \cap (\nu_i + 1)$, עבור $i < 2$. אז:

    • $\nu_0 = \min S_0 \iff \nu_1 = \min S_1$.
    • אם $\pi_{\nu_0, \nu_1}(\tau_0) = \tau_1$ ו-$\nu_0$ הוא העוקב המיידי ב-$S_0$ של $\tau_0$, אז $\nu_1$ הוא העוקב המיידי ב-$S_1$ של $\tau_1$.

הוכחה

  1. מאלמנטריות $\pi(\omega_1^{J_{\nu_0}}) = \omega_1^{J_{\nu_1}}$. כלומר $\pi(\alpha_{\nu_0}) = \alpha_{\nu_1}$.

    נוכיח ש-$\pi \vert \alpha_{\nu_0} = \text{id}$ באינדוקציה על $\beta \leq \alpha_{\nu_0}$. מאלמנטריות נובע שאם $\beta \leq \omega$ אז $\pi(\beta) = \beta$. אם $\pi(\beta) = \beta$ אזי מאלמנטריות $\pi(\beta+1) = \pi(\beta) + 1 = \beta+1$. נשאר המקרה $\beta < \alpha_{\nu_0}$ גבולי כאשר ידוע ש-$\pi \vert \beta = \text{id}$. כיון ש-$\beta < \alpha_{\nu_0} = \omega_1^{J_{\nu_0}}$ יש ב-$J_{\nu_0}$ פונקציה $f : \omega \xrightarrow{\text{לע}} \beta$. אזי $\pi(f) : \omega \xrightarrow{\text{לע}} \pi(\beta)$. לכל $n < \omega$ מתקיים $\pi(f)(\pi(n)) = \pi(f(n)) = f(n)$. כלומר $\pi(f) = f$ ולכן $\pi(\beta) = \beta$.

    $\pi$ הוא עד יחיד כיון שלשיכון אין "ברירה". נניח $x \in J_{\nu_0}$. אז $x = f(\alpha_0,\dotsc,\alpha_{n-1}, p_{\nu_0})$, כאשר $f$ נוסחה ו-$\alpha_0,\dotsc,\alpha_{n-1} < \alpha_{\nu_0}$. יש לנו $\pi(\alpha_0) = \alpha_0$, $\dotsc$, $\pi(\alpha_{n-1}) = \alpha_{n-1}$. מאלמנטריות נקבל $\pi(x) = f(\alpha_0,\dotsc,\alpha_{n-1}, p_{\nu_1})$.

  2. נניח $\nu_0, \nu_1 \unicode{x297D} \nu_2$. יהי $x \in J_{\nu_0}$. אז יש נוסחה $f(x_0,\dotsc,x_{k-1}, y)$ כך ש-$x = f(\xi_0,\dotsc,\xi_{k-1}, p_{\nu_0})$ כאשר $\xi_0,\dotsc, \xi_{k-1} < \alpha_{\nu_0}$. ואז $\pi_{\nu_0,\nu_2}(x) = f(\xi_0,\dotsc,\xi_{k-1}, p_{\nu_2})$. נקבע $\pi(x) = f(\xi_0,\dotsc,\xi_{k-1}, p_{\nu_1})$. קל לראות ש-$\pi : J_{\nu_0} \to J_{\nu_1}$ שיכון יפה, ולכן $\nu_0 \unicode{x297D} \nu_1$.

    אם $\alpha_{\nu_0} = \alpha_{\nu_1}$ אזי $\pi_{\nu_0, \nu_1}$ פונקציה על ולכן היא הזהות.

  3. איו פה כל כך מה להראות. מדובר בהרכבות של שיכונים אלמנטריים.
