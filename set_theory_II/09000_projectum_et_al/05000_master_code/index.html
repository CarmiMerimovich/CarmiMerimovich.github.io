קוד מאסטר

הגדרה $B \subseteq J_\rho$ קבוצה-$\Sigma_n^M$ תקרא קוד מאסטר-$\Sigma_n$ עבור $M$ אם מתקיים \begin{equation} \Sigma_m^{\langle J_\rho, B\rangle} = \mathscr{P}(J_\rho) \cap \Sigma_{m+n}^M, \end{equation} לכל $m < \omega$, כאשר $M$ מבנה-$J$ ו-$\rho = \rho_n^M$.
