אתר אישי

$M0'$

טענה $S$ מקיים את $M0'$.

  1. אם $\langle \alpha, \nu\rangle \in S$ אזי $\alpha < \nu < \omega_2$.
  2. אם $\langle \alpha_0, \nu_0\rangle, \langle \alpha_1, \nu_1\rangle \in S$ ו-$\alpha_0 < \alpha_1$ אזי $\nu_0 < \alpha_1$.
  3. $S_\alpha \subseteq \sup S_\alpha$ קבוצה סגורה.
  4. $\omega_1 = \max S^0 = \sup (S^0 \cap \omega_1)$ ו-$\omega_2 = \sup S^1 = \sup S_{\omega_1}$.

הוכחה

  1. כיון שב-$J_\nu$ יש מונה יחיד שאינו בן-מניה חייב להתקיים $\nu \leq \omega_2$. מהגדרת $S$ ידוע ש-$\nu$ אינו מונה ולכן $\nu < \omega_2$.

    כיון ש-$\alpha \in J_\nu$ חייב להתקיים $\alpha < \nu$.

  2. כיון ש-$\alpha_0 < \alpha_1$ נובע $J_{\nu_1} \models \ulcorner \text{הנומ וניא } \alpha_0 \ \urcorner$, ולכן $\nu_0 < \nu_1$. אם $\nu_0 > \alpha_1$ אזי ב-$J_{\nu_0}$ יש שני מונים שאינם בני מניה. סתירה. ולכן $\nu_0 \leq \alpha_1$. למה $<$ מתקיים?

  3. נניח $\nu < \omega_2$ נקודת הצטברות של $S_\alpha$. מיידי ש-$\nu$ גבול של סודרים שאינם ברי-הטלה. גם מיידי ש-$\alpha$ הוא המונה היחיד שאינן בן-מניה ב-$J_\nu$.

  4. כיון שהקבוצה $\set{N \cap \omega_2 \mid N \prec L_{\omega_2},\ N \cap \omega_2 \in \text{On}}$ מכילה סל"ח ברור ש-$\sup S_{\omega_1} = \omega_2$.

    לסיום. יהי $N \in S_{\omega_1}$ מינימלי. ניקח סדרה $\in$-עולה של תת-מודלים אלמנטריים בני מניה $\langle X_\xi \prec J_{\omega_2} \mid X_\xi \cap \omega_1 \in \omega_1,\ \xi < \omega_1\rangle$ כך ש-$N \in X_0$. המיטוט הטרנזיטיבי של כל אחד מהמודלים בסדרה $\langle N \cap X_\xi \mid \xi < \omega_1\rangle$ מוכיח ש-$\sup (S^0 \cap \omega_1) = \omega_1$.
