הסדר ⊱ על הסבך

הגדרה נניח $\nu_0, \nu_1 \in S^1$. נקבע ש-$\nu_0 \mathrel{\unicode{x22B0}} \nu_1$ אם מתקיימים הדברים הבאים:

  • $\nu_0 \neq \nu_1$.

  • $\rho(\nu_0) = \nu_0 \iff \rho(\nu_1) = \nu_1$.

  • יש שיכון $\pi : \mathfrak{a}_{0} \xrightarrow{\Sigma_1} \mathfrak{a}_{1}$ כך ש-$\pi : J_{\nu_0} \xrightarrow{Q} J_{\nu_1}$. (יש לנו הרמה יחידה $\pi: J_{\beta(\nu_0)} \xrightarrow{\Sigma_{n+1}} J_{\beta(\nu_1)}$.) בפרט מתקיימים הדברים הבאים:

    • $\pi \mid \alpha_{\nu_0} = \text{id}$.
    • $\pi(\alpha_{\nu_0}) = \alpha_{\nu_1}$.
    • $\pi(p(\nu_0)) = p(\nu_1)$.
    • אם $\nu_0 < \rho(\nu_0)$ אז $\pi(\nu_0) = \nu_1$.

שיכון $\pi$ כנ"ל נקרא יפה.
