אתר אישי

M0

טענה

  1. נניח $\langle \alpha, \nu\rangle \in S$. אזי $\alpha < \nu < \omega_2$ והסודרים $\alpha$ ו-$\nu$ סגורים תחת פונקציות רקורסיביות פרימיטיביות.
  2. נניח $\langle \alpha_0, \nu_0\rangle, \langle \alpha_1, \nu_1\rangle \in S$ ו-$\alpha_0 < \alpha_1$. אזי $\nu_0 < \alpha_1$.
    • $\omega_1 = \max S^0 = \sup (S^0 \cap \omega_1)$.
    • $\omega_2 = \sup S^1 = \sup S_{\omega_1}$.

הוכחה

  1. אם $\nu > \omega_2$ אזי $\omega_1, \omega_2 \in J_\nu$ ואז יש לנו לפחות שני מונים שאינם בני-מניה ב-$J_\nu$, ולכן $\nu \notin S^1$.

    כיון ש-$\nu \leq \omega_2$ אזי $\alpha \leq \omega_1$. כל אבר ב-$J_\nu$ אמור להיו גדיר עם פרמטרים מ-$\alpha \cup \set{p}$, ויש פה פחות מ-$\omega_2$ פרמטרים ולכן $\mid J_\nu \mid < \omega_2$ ולכן $\nu < \omega_2$.

    כיון ש$\alpha \in J_\nu$ ברור ש-$\alpha < \nu$.

  2. המשמעות של $\alpha_0 < \alpha_1$ היא ש-$\omega_1^{J_{\nu_0}} < \omega_1^{J_{\nu_1}}$. אם $\nu_0 = \nu_1$ אזי $\omega_1^{J_{\nu_0}} = \omega_1^{J_{\nu_1}}$, ולכן $\nu_0 \neq \nu_1$. אם $\nu_0 > \nu_1$ אזי $\omega_1^{J_{\nu_0}} \geq \omega_1^{J_{\nu_1}}$, ולכן $\nu_0 < \nu_1$.

    אם $\nu_0 > \alpha_1$ אזי ב-$J_{\nu_0}$ בנוסף ל-$\alpha_0$ יש עוד מונה $\alpha_1$ ולכן $\nu_0$ אינו דמוי-$\omega_2$.

  3. מסעיף 1 נובע ש-$\sup S^1 \leq \omega_2$. נראה $\sup S^1 = \omega_2$.

    יהי $\beta < \omega_2$. נקח $\beta \leq \beta' < \omega_2$ כך ש-$\beta'$ סגור תחת זוגות גדל. נקבעי $X = h_{J_{\omega_2}}'' (\omega \times \beta')$. אז $X \prec J_{\omega_2}$. תהי $\pi : X \simeq L_\gamma$ ההפלה הטרנזיטיבית. אזי $\gamma > \beta$ ו-$\gamma$ יפה.

    ביחס ל-$S^0$ נעבוד כלהלן. ראשית אם-$\nu \in S^1$ אזי $\alpha_\nu \leq \omega_1$, ולכן $\sup S^0 \leq \omega_1$. נקבע $X = h_{J_{\omega_2}}'' (\omega \times (\omega_1 + 1))$. אז $X \prec J_{\omega_2}$ ו-$X \supseteq \omega_1 + 1$. תהי $\pi : X \simeq L_\gamma$ ההפלה הטרנזיטיבית. אזי $\gamma > \omega_1$ ולכן $\alpha_{\gamma} = \omega_1$. כלומר $\max S^0 = \omega_1$.

    אחרון חביב. יהי $\alpha < \omega_1$. נקח תת-מודל אלמנטרי $X \prec J_{\omega_2}$ בגודל קטן מ-$\omega_1$, כך ש-$X \cap \omega_1 \supseteq \alpha$. תהי $\pi : X \simeq J_\gamma$ ההפלה הטרנזיטיבית. אזי $\omega_1^{J_\gamma} > \alpha$. ניקח $\nu \in S^1$ כך ש-$\nu > \gamma$. אז $\alpha_\nu > \alpha$, ולכן $\sup (S^0 \cap \omega_1) = \omega_1$.
