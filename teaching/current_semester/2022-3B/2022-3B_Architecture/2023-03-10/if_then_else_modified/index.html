שנוי קטן if ... then .... else ...

נניח שאנו רוצים לעשות שנוי קטן בקוד בשפת על השנוי קטן.

if (i == j)
    f = g + h + 2;      // C H A N G E
else
    f = g - h;

הקצאת אוגרים היא כמו קודם.

אפשרות 1:

    beq x22, x23, 12    // if i = j goto then-block
    sub x19, x20, x21   // i != j: else block
    beq x0, x0, 16       // skip over then block
    add x19, x20 x21
    add x19, x19, 2
    add x19,2

שימו לב למספרים בבתים של המרחק אל פקודות היעד. הוספנו פקודות מכונה אחת בלבד וכל המרחקים זזו. כלומר כל שנוי בקוד ידרוש מאיתנו ספירה מחדש של המרחקים.

בשביל השלמות נראה גם את אפשרות 2:

    bne x22, x23, 16     // if i != j goto then-block
    add x19, x20, x21   // i == j: else block
    add x19, x19, 2
    beq x0, x0, 8       // skip over else block
    sub x19, x20 x21
