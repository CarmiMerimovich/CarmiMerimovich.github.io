מבוא לזיכרון

למערכות זיכרון יש מיני תכונות שכדאי לתת להן שמות.

  • נדיף: זכרון הינו נדיף אם בהיפסק החשמל התוכן אובד. זיכרון ששומר את התוכן גם ללא חשמל נקרא זכרון לא-נדיף.
  • זכרון עבודה: לזכרון עבודה המעבד יכול לגשת לקריאה ולכתיבה.
  • זכרון איחסון: צריך תוכנה כדי לקרוא/לכתוב מזיכרון זה.
  • זיכרון מהיר: זיכרון שלא מאט את המעבד יותר מדי בגישה אליו.

היינו רוצים זכרון שהוא:

  • זיכרון עבודה
  • לא נדיף
  • מהיר
  • גדול
  • זול

לצערנו, אין לנו את הטכנולוגיה הנחוצה. לכן נשתמש בשילוב של זכרונות מסוגים שונים ואלגוריתמי חומרה ותוכנה כדי לקבל מערכת זיכרון שיש לה בקירוב טוב את כל התכונות הנ"ל.

זיכרון איחסון

יש את הסוגים הבאים כאשר

הרשימה מסודרת בסדר מהירות יורד ובמחיר לביט יורד:

  • דיסק מצב מוצק ssd - solid state disk
  • דיסק
  • cs/dvd
  • tape

כל הזכרונות הנ"ל הינם לא-נדיפים, בהכרח.

כל הזכרונות הנ"ל לא שומרים מידע לנצח. לא ידוע מה אורך החיים המדוייק של כל מדיה.

זכרון עבודה

יש לנו את הסוגים הבאים בסדר מהירות יורד.

  • זיכרון קריאה כתיבה RAM - Random Access Memory. נדיף.
  • זיכרון לקריאה בלבד ROM - Read Only Memory. זכרון זה לא נדיף.

RAM יש משני סוגים, מהמהיר לאיטי:

  • SRAM - Static RAM
  • DRAM - Dynamic RAM
