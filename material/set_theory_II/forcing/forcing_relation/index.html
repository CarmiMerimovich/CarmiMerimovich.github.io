יחס הכפיה

הגדרה יהי $\mathbb{P}$ מושג כפיה. נגדיר ברקורסיה את יחס הכפיה $\Vdash_{\mathbb{P}}$ באופן הבא:

  • $p \Vdash_{\mathbb{P}} \dot{y} \in \dot{x}$ אם הקבוצה \begin{align} \set{ q \leq p \mid \exists \langle r,\dot{z}\rangle \in \dot{x}\ q \leq r, q \Vdash_{\mathbb{P}} \dot{z} = \dot{y} } \end{align} צפופה מתחת ל-$p$.

  • $p \Vdash_{\mathbb{P}} \dot{y} \subseteq \dot{x}$ אם לכל $\langle \dot{z}, q \rangle \in \dot{y}$ הקבוצה \begin{align} \set{r \leq p \mid r \leq q \rightarrow \exists \langle \dot{t}, s \rangle \in \dot{x}\ r\leq s,\ r \Vdash_{\mathbb{P}} \dot{t} = \dot{z} } \end{align} צפופה מתחת ל-$p$.

  • $p \Vdash \dot{y} = \dot{x}$ אם $p \Vdash \dot{y} \subseteq \dot{x}$ ו-$p \Vdash \dot{y} \supseteq \dot{x}$.

  • $p \Vdash_{\mathbb{P}} \varphi_0(\dot{x}_0,\dotsc,\dot{x}_{n-1}) \wedge \varphi_1(\dot{x}_0,\dotsc,\dot{x}_{n-1}) $ אם $p \Vdash_{\mathbb{P}} \varphi_0(\dot{x}_0,\dotsc,\dot{x}_{n-1})$ ו-$p \Vdash_{\mathbb{P}} \varphi_1(\dot{x}_0,\dotsc,\dot{x}_{n-1})$.

  • $p \Vdash_{\mathbb{P}} \exists x \varphi(x, \dot{x}_0,\dotsc,\dot{x}_{n-1})$ אם הקבוצה $\set{q \leq p \mid \exists \dot{x}\ q \Vdash \varphi(\dot{x})}$ צפופה מתחת ל-$p$.

  • $p \Vdash_{\mathbb{P}} \lnot \varphi(\dot{x}_0,\dotsc,\dot{x}_{n-1})$ אם לכל $q \leq p$, $q \not\Vdash_{\mathbb{P}} \varphi(\dot{x}_0,\dotsc,\dot{x}_{n-1})$.

הערה אם ברור מההקשר על איזה כפיה מדובר אז נרשום רק $\Vdash$.

הערה אפשר להגדיר מה המשמעות של לכפות $\vee$ ו-$\forall x$, אבל, כרגיל אפשר להסיק מה צריכה להיות ההגדרה אם רוצים לשמור את המשמעות הלוגית, ובוודאי שאם רוצים לשמור את משפט הכפיה.
