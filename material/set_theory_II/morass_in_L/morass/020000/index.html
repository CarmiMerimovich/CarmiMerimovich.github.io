טענה יהי $\nu \in S^1$. אז $\rho^{n(\nu)+1}_{J_{\beta(\nu)}} \leq \alpha_{\nu} < \nu \leq \rho$.

הוכחה נסמן $\beta = \beta(\nu)$, $n = n(\nu)$, ולכל $k \leq n+1$, $\rho^k = \rho^k_{J_\beta}$, $A^k = A^k_{J_\beta}$ ו-$p^k = p^k_{J_\beta}$.

ראשית נראה ברקורסיה שלכל $k \leq n$, $\nu \leq \rho^k$.

המקרה $k = 0$ מיידי: $\nu \leq \beta =\rho^0$ וסיימנו.

יהי $0 \leq k < n$. $J_{\rho^k} = h''_{\langle J_{\rho^k}, A^k \rangle} (\omega \times (\rho^{k+1} \times \set{p^{k}}))$. בפרט יש פונקציה-$\Sigma_1^{\langle J_{\rho^k}, A^k\rangle}$ חלקית מ-$\rho^{k+1}$ על $J_{\rho^k}$. ברקורסיה ידוע ש-$\nu \leq \rho^k$ ולכן בפרט יש פונקציה-$\Sigma_1^{\langle J_{\rho^k}, A^k\rangle}$ חלקית מ-$\rho^{k+1}$ על $\nu$. פונקציה זו היא $\Sigma_{k+1}^{J_\beta}$. כיון ש-$k + 1 < n + 1$ חייב להתקיים $\nu \leq \rho^{k+1}$, אחרת נהיה בסתירה למינימליות $n$.

נשאר להראות ש-$\rho^{n+1} \leq \alpha_\nu < \nu$. מהגדרת $S$ נובע ש-$\alpha_\nu < \nu$. נניח בשלילה ש-$\alpha_\nu < \rho^{n+1}$. כיון ש-$\alpha_\nu$ מונה אחרון ב-$J_\nu$, יש ב-$J_\nu$ פונקציה חלקית $g : \alpha_\nu \xrightarrow{\text{לע}} \rho^{n + 1}$ ולכן $h \circ g : \alpha_\nu \xrightarrow{\text{לע}} \rho^n$ היא פונקציה-$\Sigma_1^{J_{\rho_n}}$ חלקית, בסתירה להגדרת $\rho_{n+1}$. $\square$

מסקנונת $J_{\rho^n} = h''_{\rho^n,A^n}(\omega \times (\alpha_\nu \times \set{p^{n + 1}}))$.
