מונים

הגדרה סודר $\kappa$ יקרא מונה אם לא קיימת פונקציה $f : \alpha \xrightarrow{\text{לע}} \kappa$, לכל סודר $\alpha < \kappa$.

הגדרה מונה $\kappa$ נקרא מונה סדיר (regular cardinal) אם כל פונקציה $f : \lambda \to \kappa$ הינה חסומה כאשר $\lambda < \kappa$.

הגדרה מונה שאינו סדיר נקרא חריג (singular cardinal).

הערה לפי ההגדרות דלעיל המונה $\aleph_0$ הינו סדיר. רק שהוא מונה סדיר די מעצבן ובהרבה משפטים שעוסקים במונים סדירים תופיע הדרישה שהמונה הסדיר גדול מ-$\aleph_0$.
