מספר זהויות בוליאניות >>> $(x+y)(\bar{x}+z)(y+z) = (x+y)(\bar{x}+z)$

$(x+y)(\bar{x}+z)(y+z) = (x+y)(\bar{x}+z)$

אפשר לטעון שהשוויון מתקיים כי הוא הדואלי לשוויון 1 שהוכחנו. אפשר גם לבנות את הנימוק הדואלי לנימוק שהשתמשנו בו בשוויון 1, וזה מה שנעשה.

נשים לב: אם $a + b = b$ אז $a b = a$, כי $a b = a(a + b) = a + ab = a$.

עכשיו נחשב את $(x+y)(\bar{x}+z) + (y+z)$. אז \begin{multline} (x+y)(\bar{x}+z) + (y+z)=
(x + y + y + z) (\bar{x} + y + z + z) = \\ (x + y + z) (\bar{x} + y + z) = (x + \bar{x})(y + z) = y + z. \end{multline} כלומר בסימון $a = (x + y) (\bar{x} + z)$ ו-$b = y + z$ קיבלנו $a + b = b$. וראינו שבמקרה כזה $a b = a$ כלומר \begin{equation} (x+y)(\bar{x}+z)(y+z) = (x+y)(\bar{x}+z). \end{equation}
